1. 基本面

公司靠什麼賺錢

聯發科（MediaTek）是全球最大的手機晶片設計公司之一，主要設計「系統單晶片（SoC → 就是把 CPU、GPU、記憶體控制器、數據機全塞進一顆晶片）」，自己不生產，交給台積電代工。

產業鏈位置：

```

ARM（IP授權）→ 聯發科（晶片設計）→ 台積電（代工製造）

↓

三星/小米/OPPO/vivo（手機品牌）→ 消費者

```

營收結構（2025年估算）

業務佔比趨勢
手機 SoC（天璣系列）約 55%穩定，中低階稱霸
智慧家居/IoT約 15%穩定成長
電源管理 IC約 10%穩定
車用晶片約 8%高速成長
雲端 ASIC（Google TPU）約 2-3%（快速攀升）新動能，2027 年可望達 20%
📝 ASIC：Application Specific Integrated Circuit，應用特定積體電路。就是「為特定用途量身打造的晶片」，例如 Google 為了 AI 計算特別訂製的 TPU，和一般通用晶片不同。

競爭優勢（護城河）

強度：強

  1. 手機 SoC 全球市占近 40%：中低階和中高階市場稱霸，高通主攻旗艦，兩強分食全球 98% 市場
  2. SerDes 技術積累：高速 IC 互連核心技術，是 ASIC 業務的關鍵 IP，競爭對手難以複製
  3. 客戶黏性高：晶片設計深度整合客戶 BSP（開機程式），更換供應商成本極高
  4. 台積電先進製程優先取得：長期大客戶，取得 3nm 和 2nm 產能有優先權
  5. Google TPU v7e + v8e 訂單：已鎖定兩代訂單，成為 ASIC 領域重要玩家

管理層品質

  • 董事長蔡明介：業界公認半導體老將，「農夫哲學」（深耕不追短線）獲市場信任
  • 執行長蔡力行：前台積電總裁，專業公信力極高
  • 2025 年法說會如實呈現 Q4 雙率下滑，沒有粉飾，誠信度佳
  • ⚠️ 單一來源：管理層品質評估來自新聞報導，建議觀察法說會實際言行一致性

雙軌估值（進場價格 SOP）

基本數據：

  • 近 4 年 EPS：74.59 / 48.51 / 66.92 / 66.16 元（平均 64.05 元）
  • 2026 年 Forward EPS 市場預估：70~72 元，取 71 元
  • 歷史 PE 區間（2023-2026）：8.81x ~ 29.48x，低標約 14x，中位數約 19x，高標約 24x

方法 A（歷史 PE 區間法，用 Forward EPS）：

```

合理價 = Forward EPS 71 × 歷史 PE 中位數 19x = 1,349 元

```

方法 B（PEG 估值法）：

```

2026 vs 2025 EPS 成長率 = (71-66.16)/66.16 = 7.3%

→ 成長率偏低，直接用 PEG=1 → 合理 PE = 7.3x（過低，加入 ASIC 題材溢價）

→ 市場共識：ASIC 轉型值得 18-20x PE

→ 取 20x（考量 ASIC 成長加速的未來溢價，上限 30x）

合理價 = Forward EPS 71 × 20x = 1,420 元

```

差距計算：

```

差距 = |1,349 - 1,420| / 1,349 = 5.3% → ≤ 20% → 取平均

錨定價 = (1,349 + 1,420) / 2 = 1,385 元

```

進場區間（TYPE_A）：

```

entry_high = 1,385 × 0.90 = 1,247 元

entry_low = 1,385 × 0.85 = 1,177 元

```

技術面微調：

  • MA120（半年線）= 1,481 元，遠高於 entry_high（1,247 元）
  • 現價 1,575 元，高於 entry_high 26%
  • → 備註：「技術支撐在 MA120 = 1,481，高於公式進場區。現價遠高於進場區，不建議追高加碼。若要加碼，等回調至 1,400-1,481 元區間（MA120 附近）再評估。」

估值結論：

```

方法 A（歷史PE法）：合理價 1,349 元（Forward EPS 71 × PE 19x）

方法 B（PEG+ASIC溢價法）：合理價 1,420 元（Forward EPS 71 × PE 20x）

差距：5.3% → 取平均

錨定價：1,385 元

進場區間：1,177 ~ 1,247 元

目前現價 1,575 元 → 溢價約 14% 高於 entry_high

備註：ASIC 轉型若如期加速，可支撐更高 PE，但估值已偏貴，持有者按兵不動

```

📝 PEG：本益比除以 EPS 成長率。PEG = 1 代表股價成長和獲利成長剛好匹配，< 1 代表成長被低估。聯發科因為 ASIC 題材，市場願意給額外溢價。

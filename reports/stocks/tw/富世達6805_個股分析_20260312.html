1. 基本面

公司簡介

富世達成立於 2001 年，是奇鋐（3017）的子公司，股本約 10.7 億元。原本是全球領先的精密轉軸（Hinge）製造商 → 就是筆電、手機螢幕可以翻開/折疊的那個金屬關節零件。

近年成功打造雙引擎成長策略：

  1. 折疊手機轉軸（第一引擎）：以「水滴型轉軸」技術切入華為、Motorola 等一線品牌
  2. AI 伺服器液冷散熱零組件（第二引擎）：生產快接頭（QDC）、浮動接頭等關鍵零件

產業鏈定位

```

上游（原料） 中游（零組件） 下游（系統/品牌）

金屬材料/CNC加工 → 【富世達】轉軸/快接頭 → 奇鋐（散熱模組）→ 雲端四大客戶（CSP）

→ 華為/Motorola（折疊手機）

```

→ 富世達在中游，做的是「小但關鍵」的零組件。跟著母公司奇鋐直接打入四大雲端業者供貨體系。

營收結構（2025 年）

業務佔比趨勢
轉軸（折疊手機+筆電）~75%穩定成長，華為三折機帶動
AI 伺服器液冷零組件~22%爆發成長（2024 年僅 5%）
其他~3%—

客戶集中度

  • 母公司奇鋐為最大客戶（伺服器零組件透過奇鋐出貨）→ 集團內關係，穩定但也意味著依賴母公司
  • 折疊手機端：華為（外折規格）、Motorola 為主力
  • ⚠️ 華為佔折疊手機營收比重高，地緣政治風險需留意

競爭優勢（護城河）

  • 精密金屬加工技術：水滴型轉軸技術門檻高，折疊次數、手感精度要求極嚴
  • 集團綜效：背靠奇鋐，直接卡位 NVIDIA GB200/GB300 液冷供應鏈
  • 客戶認證門檻：快接頭需通過 NVIDIA 認證（NVQD），新進者不易取代
  • 護城河評級：中（技術有門檻，但不是不可被追趕，且依賴母公司通路）

管理層

  • 經營者為奇鋐集團體系，管理層穩定
  • 近年成功從純轉軸廠轉型為雙引擎，策略執行力佳

雙軌估值

股票類型判定：EPS YoY > 15% → TYPE_A 成長股

方法 A（歷史 PE 區間法 — 回顧型）：

```

近 4 年平均 EPS = (9.33 + 10.18 + 17.90 + 31.48) / 4 = 17.22 元

歷史 PE 低標（25th）≈ 30x（2022-2024 年本益比區間推估）

歷史 PE 中位數 ≈ 40x

合理價 = 17.22 × 40 = 689 元

```

⚠️ 但富世達 2024 年前是小型轉軸股，2025 年才轉型爆發，歷史 PE 區間對照性不強。改用 Forward EPS：

```

Forward EPS = 40.62 元（2026E 保守）

合理價 A = 40.62 × 30（PE 低標）= 1,219 元

```

→ 方法 A 合理價：1,219 元（Forward EPS 40.62 × 歷史 PE 低標 30x）

方法 B（PEG 估值法 — 前瞻型）：

```

EPS 年成長率 = (40.62 - 31.48) / 31.48 = 29.1%（2025→2026 保守）

合理 PE = 29.1 × 1（PEG=1）= 29.1x → 取 29x

Forward EPS = 40.62 元

合理價 B = 29 × 40.62 = 1,178 元

```

→ 方法 B 合理價：1,178 元（PEG=1，Forward PE 29x × EPS 40.62）

差距判定：

```

差距 = |1219 - 1178| / 1178 = 3.5% → ≤ 20% → 取平均

錨定價 = (1219 + 1178) / 2 = 1,199 元（取整 1,200）

```

進場區間（TYPE_A）：

```

entry_high = 1,200 × 0.90 = 1,080 元

entry_low = 1,200 × 0.85 = 1,020 元

```

技術面微調：MA120（半年線）在 1,397 元，遠高於公式值。現價 1,840 遠高於 entry_high 超過 70%。

⚠️ 重要備註：

  • 此估值使用保守預估 EPS 40.62 元。若用凱基樂觀預估 58.49 元，合理價會顯著上升
  • 方法 A 若用 58.49 × 30 = 1,755 元、方法 B 若用 EPS 成長率 86%（但 PE 上限 30x）= 30 × 58.49 = 1,755 元
  • 樂觀情境錨定價約 1,755 元，進場區間 1,492~1,580 元
  • 保守 vs 樂觀差距大，反映市場對伺服器放量速度的分歧
  • 現價 1,840 元已超過樂觀情境的合理價，短期偏貴

1. 基本面

公司簡介

智邦科技（股票代號：2345）是台灣的網通設備 ODM/JDM 大廠。

  • ODM（Original Design Manufacturer）：幫客戶設計加製造，產品不掛自己品牌
  • JDM（Joint Design Manufacturer）：客戶提出需求規格，智邦從設計到出貨全包，深度共同開發

核心產品是乙太網路交換器（→ 資料中心裡面負責讓伺服器互相溝通的「高速交通管理員」），以及近年快速成長的 AI 加速卡模組。

一句話說清楚：智邦幫全球最大的雲端公司（AWS、Google、Meta 等）做 AI 資料中心裡的「高速公路交匯處」，並且在這個賽道全球市占率近五成。

產業鏈定位

```

上游：網通晶片（Broadcom、Marvell）、PCB、被動元件

↓

中游：【智邦】→ 設計 + 製造交換器、AI 加速卡模組

↓

下游：超大規模雲端客戶（AWS、Google、Meta、Microsoft）、電信商、企業

```

雙軌估值

方法 A：歷史 PE 區間法（回顧型）

```

Forward EPS（2026E）估算：

法人預估智邦 2026 年 EPS = 56.3 元

（來源：美系外資報告，以 2025 全年 47.13 × 預估成長率約 20% 估算）

歷史 PE 區間（近 3 年）：

  • PE 低標 ≈ 25x
  • PE 中位數 ≈ 32x
  • PE 高標 ≈ 45x

fair_value_a = Forward EPS 56.3 × 歷史 PE 中位數 32x = 1,802 元

```

⚠️ Forward EPS 56.3 元為美系外資報告推估值，非公司官方數據

方法 B：PEG 估值法（前瞻型）

```

2025 EPS 成長率 = 119%

2026 EPS 成長率預估 = 20%（法人共識，基期墊高後放緩）

→ 未來 2 年平均成長率 ≈ 50%

合理 PE = 成長率（%）× 1（PEG = 1）= 50x

但 TYPE_A 上限 30x → 取 30x

fair_value_b = 30x × 56.3 = 1,689 元

```

差距判定

```

差距 = |1,802 - 1,689| / min(1,802, 1,689)

= 113 / 1,689 = 6.7%

→ 差距 < 20% → 取兩者平均

anchor = (1,802 + 1,689) / 2 = 1,746 元

anchor_rule = "差距<20%取均值"

```

進場區間（TYPE_A：anchor × 0.85 ~ 0.90）

```

entry_high = 1,746 × 0.90 = 1,571 元

entry_low = 1,746 × 0.85 = 1,484 元

```

#風險發生機率影響程度早期警訊可控？
1客戶集中風險：AWS 佔營收比過高，若大幅縮減訂單或轉單低致命AWS 法說會下修 CAPEX 指引不可控

致命風險提醒

AWS 大幅砍單和全球 AI 資本支出斷崖下降是兩個致命風險。目前機率低，但一旦發生，股價腰斬的機率很高。需持續追蹤 AWS（Amazon）、Google Cloud、Meta 每季法說會的 CAPEX 數字。

關稅風險新增（2026 年）：美國針對網通設備的關稅政策動向需要特別留意，雖然智邦已佈局多地產線分散，但短期過渡期仍可能衝擊毛利。

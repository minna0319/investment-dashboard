1. 基本面

公司簡介

富世達成立於 2001 年，是奇鋐（3017）的子公司，股本約 10.7 億元，上市代號 6805.TW。原本是全球領先的精密轉軸（Hinge）製造商 → 轉軸就是筆電螢幕、折疊手機可以翻開/折疊的那個金屬關節零件。

近年成功打造雙引擎成長策略：

  1. 折疊手機轉軸（第一引擎）：以「水滴型轉軸」技術切入華為、Motorola 等一線品牌
  2. AI 伺服器液冷散熱零組件（第二引擎）：生產快接頭（QDC / NVQD / UQD）、浮動接頭、滑軌等關鍵零件

→ NVQD = NVIDIA 認證的專用快接頭規格，供應商必須通過 NVIDIA 認證才能出貨

產業鏈定位

```

上游（原料） 中游（零組件） 下游（系統/品牌）

金屬材料 / CNC 加工 → 【富世達】轉軸 / 快接頭 → 奇鋐（散熱模組）→ 雲端四大 CSP

→ 華為 / Motorola（折疊手機）

```

→ 富世達在中游，做「小但關鍵」的精密零組件。透過母公司奇鋐，直接打入 NVIDIA GB200 / GB300 / Rubin 平台液冷供應鏈。

  • 折疊手機：華為（外折 / 三折規格）、Motorola 為主力
  • ⚠️ 華為佔折疊手機業務比重高，地緣政治風險需持續關注

競爭優勢（護城河）

  • 精密金屬加工技術：水滴型轉軸技術門檻高，折疊次數、手感精度要求嚴苛
  • 集團綜效：背靠奇鋐，直接卡位 NVIDIA 液冷供應鏈
  • NVQD 客戶認證門檻：快接頭需通過 NVIDIA 認證，新進者短期不易取代
  • 護城河評級：中（技術有門檻，但仰賴母公司通路，且為 Tier 2 供應商）

管理層

  • 經營者為奇鋐集團體系，管理層穩定
  • 近年成功從純轉軸廠轉型為雙引擎，策略執行力佳
  • 2025 法說會（10/19）公開說明 Rubin 平台認證進度，資訊透明度中等

→ 股價淨值比（PBR）= 股價 / 每股帳面淨值，17.82 代表市場願意付出淨資產 17 倍的溢價

雙軌估值（基於 2026E 保守 EPS 40.62 元）

股票類型判定：EPS YoY > 15% → TYPE_A 成長股

方法 A（歷史 PE 區間法）：

```

2026E EPS = 40.62 元

歷史 PE 低標 = 30x（近 3 年估值低檔參考）

合理價 A = 40.62 × 30 = 1,219 元

```

⚠️ 富世達 2025 年前為小型轉軸股，轉型後本益比體系改變，歷史 PE 對照性有限

方法 B（PEG 估值法 → 用成長率決定合理 PE）：

```

EPS 成長率（2025→2026 保守）= (40.62 - 31.00) / 31.00 = 31.0%

合理 PE = 31.0 × 1（PEG = 1）= 31x

合理價 B = 40.62 × 31 = 1,259 元

```

→ PEG = 1 = 成長率和 PE 相同，是市場認為「合理定價」的標準線

差距判定：

```

差距 = |1219 - 1259| / 1219 = 3.3% → ≤ 20% → 取平均

錨定價 = (1219 + 1259) / 2 = 1,239 元（取整 1,240）

```

進場區間（TYPE_A）：

```

entry_high = 1,240 × 0.90 = 1,116 元

entry_low = 1,240 × 0.85 = 1,054 元

```

技術面參考：MA60（季線）在 1,651.5 元，遠高於公式值。現價 1,870 元高於 entry_high 約 67%。

⚠️ 重要備註：

  • 此估值採保守 EPS 40.62 元。若採摩根士丹利樂觀情境（利潤年增 60% → EPS ~50 元），合理價 A 約 1,500 元，進場區間升至 1,275~1,350 元
  • 現價 1,870 元仍顯著高於樂觀估值的進場區間，估值偏高
  • 目前市場給予的 PE 60x，隱含市場認為 2026 EPS 可達 31 元，對應 2027 仍有高成長預期 → 一旦成長不及預期，修正幅度大

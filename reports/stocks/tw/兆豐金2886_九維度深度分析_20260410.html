1. 基本面

公司是什麼

兆豐金融控股（2886）是台灣資產規模最大的公股金控，背後有政府股份支撐。旗下最重要的子公司是兆豐銀行，貢獻整體獲利約 84%。

📝 公股金控 → 就是政府持有大量股份的金融公司。好處是倒閉機率極低、監管嚴格；缺點是獲利成長比民營金控保守。

業務結構

```

兆豐金融控股

├── 兆豐銀行（獲利貢獻 ~84%）← 核心

│ ├── 國內放款（約 71%）

│ └── 外幣/海外放款（約 29%）

├── 兆豐票券（貢獻 ~7%）

├── 兆豐證券（貢獻 ~4%）

└── 其他（保險、資管、創投）

```

海外布局

兆豐銀行在全球有 39 個海外據點，橫跨 18 個國家，以美洲、東南亞為主（新南向政策配合 22 個據點在新南向區域）。這是台灣金控中海外布局最廣的之一。

  • 總體而言，管理層無重大誠信疑慮，但決策偏保守（公股特性）

3. 估值面

股票類型判定

殖利率 5.84% > 3%，EPS 成長率 < 15% → TYPE_B（價值/存股型）

雙軌估值方法：方法 A（歷史 PE 區間法）+ 方法 B（股利折現法）

方法 A：歷史 PE 區間法

近 4 年 EPS：2.36（2025）、2.35（2024）、2.37（2023）、1.32（2022）

說明：2022 年 EPS 受金融市場動盪拖累明顯偏低，屬非常態年份。採近 3 年平均更能反映目前獲利能力（2022 年為升息衝擊下的一次性低點）。

近 3 年平均 EPS = (2.36 + 2.35 + 2.37) / 3 = 2.36 元

歷史 PE 區間（⚠️ 單一來源，引用市場共識數據）：

  • 台灣金融股 PE 歷史區間約 12~20 倍，公股金控常落在 14~18 倍
  • 近期 PE（TWSE）：14.73 倍（接近低標）
  • 依市場共識：低標 ~13x，中位數 ~16x，高標 ~19x

方法 A 合理價 = 2.36 × 16 = 37.8 元

方法 A 公式：近 3 年平均 EPS 2.36 × 歷史 PE 中位數 16x

方法 B：股利折現法

近 3 年平均現金股利 = (1.60 + 1.50 + 1.24) / 3 = 1.447 元

目標殖利率 = 近 3 年平均殖利率 = (3.72% + 3.61% + 3.10%) / 3 = 3.48%

方法 B 合理價 = 1.447 ÷ 3.48% = 41.6 元

方法 B 公式：近 3 年平均現金股利 1.447 ÷ 近 3 年平均殖利率 3.48%

差距判定與進場區間

```

差距 = |37.8 - 41.6| / 37.8 = 10.1% → ≤ 20%，取平均

錨定價 anchor = (37.8 + 41.6) / 2 = 39.7 元

取值規則：取平均

```

TYPE_B 進場區間：

  • entry_high = 39.7 × 1.00 = 39.7 元
  • entry_low = 39.7 × 0.95 = 37.7 元
  • 標準寬度：5.0%（符合規範 3-5%）

現價 39.85 元，略高於 entry_high 39.7 元（高出約 0.4%）。

→ 備註：現價幾乎恰在合理價上緣，已持有者安心持有；若要加碼，等回調至 37.7-39.7 元區間

📝 股利折現法 → 如果你把未來每年的配息折算成現在的價值，股票應該值多少錢。就像你問「如果這張股票每年配 1.45 元，我要求 3.48% 的報酬率，那它值多少？」

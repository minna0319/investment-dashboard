1. 基本面

公司一句話介紹

研華是全球工業電腦（IPC）龍頭，專做讓工廠、醫院、交通、零售等場所「變聰明」的嵌入式電腦設備與 AIoT 軟體平台。

工業電腦（IPC） → 不是一般家用電腦，而是能在灰塵多、溫差大、震動劇烈的工廠或戶外環境長期穩定運作的電腦系統。

主要業務與營收結構

事業群業務內容佔比（估）趨勢
嵌入式物聯網（EIoT）工業主機板、嵌入式電腦、機器視覺~40%穩定成長
工業自動化（iAS）工業控制器、HMI 人機介面~25%回溫中
服務型物聯網（SIoT）AIoT 平台 WISE-IoT、解決方案~20%穩定
邊緣 AI（Edge AI）NVIDIA 晶片邊緣運算設備~17%（2025 Q3）高速成長
Edge AI（邊緣 AI） → AI 不跑在遠端雲端，而是直接在現場設備上即時處理數據。工廠的機械手臂即時辨識瑕疵品，就是邊緣 AI 的應用。

產業鏈位置

```

上游（晶片） 中游（系統設計製造） 下游（終端應用）

NVIDIA / Intel → 研華（設計+製造+軟體） → 工廠/醫院/零售/交通

AMD / 記憶體廠 樺漢/凌華（競爭者） 系統整合商/最終用戶

```

競爭優勢（護城河）

  1. 全球市佔第一：工業電腦市佔約 42.5%，規模遙遙領先第二名
  2. 高客戶轉換成本：工業電腦嵌入客戶產線，換掉需重新認證、測試，成本極高
  3. 產品線最齊全：從硬體到軟體到解決方案一條龍
  4. NVIDIA 深度合作：2026 NVIDIA GTC 大會展示 NVIDIA Jetson Thor 驅動的邊緣 AI 設備（ASR-A702、MIC-742），為 Edge AI 領域首選合作夥伴
  5. 全球化佈局：在全球 27 國設有據點，具規模經濟優勢
  6. Physical AI 新方向：2026 年開始佈局機器人（AMR、人形機器人）的 GMSL 相機方案，與 NVIDIA Isaac ROS 深度整合
護城河 → 讓競爭對手很難搶走生意的優勢。研華的護城河在於全球第一的市佔、客戶黏著度高，加上與 NVIDIA 的深度合作關係。

管理層

  • 創辦人劉克振：1983 年創辦，業界公認的 IPC 教父，經營超過 40 年
  • 共治總經理陳清熙、張家豪：2026 年 3 月法說會由三人共同主持
  • 管理層穩定，近年積極推動 Edge AI + Physical AI 雙軌轉型，策略方向清晰

基本面評分：強（護城河強，轉型方向明確，管理層穩定）

雙軌估值計算

方法 A：歷史 PE 區間法（回顧型）

```

近 4 年平均 EPS ≈ (13.87 + 12.99 + 10.45 + 12.25) / 4 = 12.39 元

歷史 PE 低標（25th percentile）≈ 23x

歷史 PE 中位數 ≈ 28x

歷史 PE 高標（75th percentile）≈ 33x

便宜價 = 12.39 × 23 = 285 元

合理價 = 12.39 × 28 = 347 元

昂貴價 = 12.39 × 33 = 409 元

fair_value_a = 347 元

```

方法 B：PEG 估值法（前瞻型）

```

Forward EPS 2026（法人中位數估）≈ 14.2 元

EPS 年成長率 ≈ 16%（2025 實際 +17.2%，取保守 16%）

合理 PEG = 1.2（龍頭有溢價空間）

合理 PE = 16 × 1.2 = 19.2x → 偏低，以歷史低標 23x 為底

fair_value_b = 23 × 14.2 = 327 元

```

差距判定與進場區間

```

差距 = |347 - 327| / 347 = 5.8% → ≤ 20% → 取平均

anchor = (347 + 327) / 2 = 337 元

TYPE_A 進場區間（anchor × 0.90 ~ × 0.85）：

entry_high = 337 × 0.90 = 303 元

entry_low = 337 × 0.85 = 286 元

技術面微調：MA60 = 320 → 吸附至 320，作為進場區間修正上緣

最終進場區間：286 ~ 320 元

```

估值結論

  • 現價 366 元高於錨定價 337 元約 +8.6%
  • 現價高於進場區間上緣（320）約 +14.4%
  • 目前估值合理偏貴，尚未到「明顯超貴」，但也非低估
PEG → 把「成長速度」也考慮進本益比，PEG = 1 代表股價成長跟獲利成長剛好對應，< 1 偏便宜，> 1 偏貴。

1. 基本面

公司一句話介紹

研華是全球工業電腦（IPC）龍頭，專門做「讓工廠、醫院、交通、零售等場所變聰明」的電腦設備和軟體平台。

主要業務

  • 工業電腦硬體：嵌入式電腦、工業主機板、人機介面、工業伺服器
  • 邊緣 AI（Edge AI）：搭載 NVIDIA 晶片的邊緣運算設備，讓 AI 可以在現場（而非雲端）即時處理數據
  • 軟體平台（WISE-IoT）：物聯網管理平台，提供數據收集、分析、可視化
  • 系統整合服務：針對不同產業（製造、醫療、零售、交通）提供完整解決方案

營收結構（2025 年）

事業群佔比（估）趨勢
嵌入式物聯網（EIoT）~40%穩定成長
工業自動化（iAS）~25%回溫中
服務型物聯網（SIoT）~20%穩定
邊緣 AI 相關~15%高速成長（2025 Q1 達 15.1%，年底估 20%+）

產業鏈定位

```

上游（晶片/零組件） 中游（系統設計製造） 下游（終端應用）

Intel / NVIDIA → 研華（設計+製造） → 工廠/醫院/零售/交通

AMD / 記憶體廠 凌華/樺漢（競爭者） 系統整合商/最終用戶

```

競爭優勢（護城河）

  1. 全球市佔第一：工業電腦市佔約 42.5%，在日、韓、美幾乎沒有直接競爭者
  2. 客戶轉換成本高：工業電腦嵌入客戶的產線/系統，換掉要重新認證、測試，成本極高
  3. 產品線最齊全：從硬體到軟體到解決方案一條龍，競爭對手多只做其中一段
  4. 與 NVIDIA 深度合作：唯一受邀出席 NVIDIA 供應鏈宴會的 IPC 廠商，Edge AI 領域的首選合作夥伴
  5. 全球化佈局：在全球 27 國設有據點，具規模經濟優勢

管理層

  • 創辦人劉克振：1983 年創辦研華，經營超過 40 年，產業界公認的 IPC 教父級人物
  • 管理團隊穩定，近年積極推動 Edge AI 轉型，策略方向清晰
  • 過去法說會的營運目標達成率良好
護城河 → 讓競爭對手很難搶走生意的優勢。研華的護城河在於全球第一的市佔、客戶黏著度高（換掉研華的設備太麻煩了），加上跟 NVIDIA 的獨家合作關係。

```

```

說明：研華近年 PE 區間大約在 23~33 倍之間，中位數約 28 倍。用近 4 年平均 EPS 計算。

方法 B（PEG 估值法 — 前瞻型）

```

Forward EPS（2026 法人中位數）≈ 14.4 元

EPS 年成長率 ≈ 20%（取中位預估）

合理 PE = 20 × 1（PEG=1）= 20x

但 PE 下限以歷史低標 23x 為底

fair_value_b = 23 × 14.4 = 331 元

（若用 PEG=1 的 20x → 14.4 × 20 = 288，但這偏低估了龍頭溢價）

取合理 PEG=1.2 → PE = 24x → 14.4 × 24 = 346 元

fair_value_b = 346 元

```

本益比（PE） → 用現在的股價除以每股盈餘，代表「你願意花幾年的獲利來買這檔股票」。PE 28 倍 = 用 28 年的獲利來買，聽起來很久，但如果公司成長快，實際回本會比這快。
PEG → 「考慮成長性之後，這檔股票到底貴不貴」。PEG = 1 表示股價成長跟獲利成長剛好匹配，< 1 偏便宜，> 1 偏貴。

1. 基本面

公司簡介

智邦科技是台灣的網通設備 ODM/JDM 大廠（ODM → 幫客戶設計加製造，JDM → 客戶提需求、智邦從設計到出貨全包），核心產品是乙太網路交換器（→ 資料中心裡面負責讓伺服器互相溝通的設備）。

一句話說清楚：智邦幫全球最大的雲端公司（AWS、Google、Meta 等）做 AI 資料中心裡的「高速公路交匯處」。

產業鏈定位

```

上游：網通晶片（Broadcom、Marvell）、PCB、被動元件

↓

中游：【智邦】→ 設計 + 製造交換器、AI 加速卡模組

↓

下游：超大規模雲端客戶（AWS、Google、Meta、Microsoft）、電信商、企業

```

雙軌估值

方法 A：歷史 PE 區間法（回顧型）

```

近 4 年平均 EPS = (5.83 + 11.28 + 21.49 + 44.27) / 4 ≈ 20.72 元

（注：2022=5.83, 2023=11.28, 2024=21.49, 2025E=44.27）

但智邦正處高速成長期，過去 EPS 已不代表未來

→ 改用 Forward EPS = 44.27 元（2025 法人預估中位數）

歷史 PE 區間（近 3 年）：

  • PE 低標（25th percentile）≈ 25x
  • PE 中位數 ≈ 32x
  • PE 高標（75th percentile）≈ 40x

fair_value_a = Forward EPS 44.27 × 歷史 PE 中位數 32x = 1,417 元

```

方法 B：PEG 估值法（前瞻型）

```

2025 年 EPS 成長率 ≈ 106%（從 21.49 到 44.27）

2026 年 EPS 成長率預估 ≈ 30-40%（法人共識，營收成長趨緩但基數高）

→ 取未來 2 年平均成長率 ≈ 50%

合理 PE = EPS 年成長率 × 1（PEG = 1）= 50x

但 TYPE_A 上限 30x → 取 30x

Forward EPS ≈ 44.27 元（2025E）

→ 若取 2026 Forward EPS，法人預估約 55-60 元（以 30% 成長計算 ≈ 57.6 元）

fair_value_b = 30x × 57.6 = 1,728 元

```

注意：方法 B 使用的 2026 Forward EPS 57.6 元為推估值，非官方數據，標註「⚠️ 推估值」

差距判定

```

差距 = |1,417 - 1,728| / min(1,417, 1,728) = 311 / 1,417 = 21.9%

→ 21-40% → 取較低者

anchor = 1,417 元

anchor_rule = "取較低(方法A)"

```

進場區間（TYPE_A：anchor × 0.85 ~ 0.90）

```

entry_high = 1,417 × 0.90 = 1,275 元

entry_low = 1,417 × 0.85 = 1,204 元

```

  • MA120（半年線）= 1,122 元，遠低於進場區間

  • 現價 1,505 元遠高於 entry_high 1,275 元（+18%）→ 標註「等回調」

比較結論

  • 智邦 vs Celestica：同為白牌交換器代工，但智邦市占率遠高於 Celestica，技術領先（800G/1.6T），在 AI 賽道卡位更好。

  • 在同業中的相對劣勢：毛利率最低（代工模式的先天限制）

催化劑解讀

  • 800G → 1.6T 的規格升級是最核心的成長引擎。就像手機從 4G 升級到 5G，每次交換器規格翻倍，單台價格就翻好幾倍，智邦的營收就會跟著跳升
  • L11 機架級整合：下一代伺服器架構中，交換器從「可有可無」變成「必備核心」，這會大幅提升智邦的產品價值
  • 新客戶拓展：降低對 AWS 的依賴，是長期正面因素

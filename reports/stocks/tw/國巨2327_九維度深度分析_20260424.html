1. 基本面

一句話說清楚

國巨是全球前三大「被動元件」廠，靠賣每個電子產品裡都需要的小零件（電容、電阻、電感）維生，規模龐大、產品線最全，AI 伺服器讓這些小零件需求量暴增、售價攀升，正在享受結構性漲價的紅利。

📝 被動元件：電路板上不需要電源主動運作的小零件（對比有 CPU、GPU 這種「主動」晶片）。每台智慧手機有數百到上千顆，每台 AI 伺服器用量是傳統伺服器的 2~3 倍，量大、種類多、門檻低，但是全球通用、離不開它。

產業鏈定位

```

上游原料（銀粉、鋇鈦氧化物、鉭粉、鎳）

↓

【國巨】中游被動元件製造 ← 你在這裡

↓

下游品牌客戶（蘋果、NVIDIA、車廠、工業設備商）→ 終端產品

```

主要業務與產品線

國巨靠多次併購建構起台廠中最完整的被動元件產品矩陣：

產品類別來源主要應用趨勢

📝 MLCC（積層陶瓷電容）：Multiple Layer Ceramic Capacitor，多層陶瓷電容器。就是電路板上用來「穩定電壓、濾除雜訊」的小方塊。一台 NVIDIA GB200 AI 伺服器機櫃需要高達 44 萬顆，比傳統伺服器多出近 20 倍。
📝 TLVR 電感：Total Loss Voltage Regulator 電感，專為 AI 處理器（GPU）供電穩定設計的高規格電感，耐高電流、高頻率，是 AI 伺服器的必備零件。
📝 鉭電容（Tantalum Capacitor）：用金屬鉭當介質的電容器，比 MLCC 更耐高溫、電容值更穩定。因為鉭很稀有且產地集中（剛果、盧安達），價格高、產量少，主要用在國防軍工、航太、高階車用這種「不能壞」的場景。國巨 2020 年併購美國 KEMET 就是切進這塊高門檻市場。

3. 估值面

股票類型判定

  • 2026 年 EPS 年增預估約 +40%（2025 年 11.51 → 2026E 16.1 元）
  • 但被動元件是景氣循環股，週期頂點 EPS 高、PE 反而被壓縮
  • 判定：景氣循環型成長股（Cyclical Growth），估值邏輯特殊，見下方說明
📝 景氣循環股：獲利跟著產業景氣起伏的公司。被動元件每幾年就會因為供過於求或供不應求而大幅波動。有趣的是，景氣循環股在獲利高峰時 PE 反而低（市場知道好日子不會太久），在獲利谷底時 PE 反而高（市場期待未來改善）。這和一般成長股的估值邏輯相反。

景氣循環股估值邏輯補充：被動元件多頭週期中，市場不會用歷史高 PE 定價，而是看「週期還剩多少」。法人更傾向看 EV/EBITDA 或絕對目標價，而不是直接套本益比。本分析沿用雙軌 PE 法，但讀者需理解這是保守框架，不完全適用於景氣循環股高點。

雙軌估值

基本數據：

  • 2025 年 EPS：11.51 元（已公布）
  • 2026E EPS：16.1 元（法人最新共識，FactSet 最新調查 10.68 元為保守端）
  • 保守估 Forward EPS：取法人共識 16.1 元與 FactSet 保守值 10.68 元的中值 ≈ 13 元（⚠️ 保守假設）
  • 歷史 PE 區間（2023-2026，拆股後重建）：低標約 9-12x，中位數約 18-22x，高標約 26-30x

方法 A（歷史 PE 區間法）：

```

合理價 = Forward EPS 13 × 歷史 PE 中位數 20x = 260 元

```

方法 B（被動元件景氣中段估值，PE 18-22x）：

```

景氣循環股不宜用 PEG，改用產業週期合理 PE 區間

當前被動元件景氣週期：供應緊缺中段（非高峰），村田/三星電機稼動率 90-95%

合理 PE 取 22x（景氣中段，漲價持續）

合理價 = 13 × 22x = 286 元

```

差距計算：

```

差距 = |260 - 286| / 260 = 10% → ≤ 20% → 取平均

錨定價 = (260 + 286) / 2 = 273 元

```

進場區間：

```

entry_high = 273 × 0.90 = 245.7 → 取 246 元

entry_low = 273 × 0.85 = 232.1 → 取 232 元

```

技術面微調：

  • MA60 = 277.26 元，高於 entry_high（246 元）
  • 現價 297 元高於 entry_high 21%
  • → 備註：技術支撐在 MA60（277）和 MA20（287）附近，高於公式進場區。若要等進場，可觀察回調至 MA60（277）附近的機會，而非死等公式下緣。

估值結論：

```

方法 A（歷史 PE 區間法）：合理價 260 元（Forward EPS 13 × 歷史 PE 中位數 20x）

方法 B（景氣循環週期法）：合理價 286 元（Forward EPS 13 × 景氣中段 PE 22x）

差距：10% → 取平均

錨定價：273 元

進場區間：232 ~ 246 元

現價 297 元 → 已超出 entry_high 約 21%，偏高

```

⚠️ 景氣循環股特別警示：如果 2026E EPS 真的達到法人樂觀預估的 16.1 元，Forward PE = 297/16.1 = 18.4x，現價並不算貴。但如果景氣在 H2 2026 出現轉折，EPS 到頂後下修，PE 反而會急劇收縮。景氣循環股的致命風險正是「EPS 越高、股價越危險」的悖論。

1. HVDC 架構帶動新一輪被動元件需求

  • AI 伺服器為應對龐大耗電與散熱，走向高壓直流（HVDC）架構
  • 驅動鋁電容、鉭電容（SPCAP/TCAP）、TLVR 電感、高階 MLCC 需求大增
  • 對國巨直接利多：KEMET 鉭電容 + 磁性元件（Pulse TLVR 電感）雙引擎同時受惠
📝 HVDC（高壓直流）：High Voltage Direct Current。傳統伺服器用低壓交流配電，HVDC 則是高壓直流直送晶片，減少轉換損耗。AI 伺服器耗電高、HVDC 能大幅降低能耗，是下一代資料中心主流架構。

>

📝 SPCAP / TCAP：兩種高階鉭電容規格（Solid Polymer / Tantalum Capacitor），體積小、穩定性高，用在伺服器電源管理。

2. 日系轉單效應（產能排擠）

  • 日本三大廠（村田 / 太陽誘電 / TDK）將產能全部往 AI 與車用高階集中
  • 中低階 / 消費型被動元件產能被釋出 → 台廠（含國巨）接到轉單
  • 類似先前記憶體市場的排擠效應，是結構性而非短期題材

3. 稼動率 = 進場時機的觀察指標

  • 股癌進場邏輯：觀察稼動率（產能利用率）慢慢墊高
  • 當稼動率滿載時，下游怕拿不到貨會出現「包產能」行為
  • 包產能一發生，報價會瞬間暴衝 → 此時漲價題材進入高潮

⚠️ 致命風險

  1. 景氣循環反轉：被動元件是高度週期性產業，2019 年和 2022 年都曾出現 EPS 暴漲後快速崩落的案例。一旦 AI 資本支出縮減或大廠擴產，漲價題材可能迅速退潮，EPS 從 16 元反轉至個位數也不令人驚訝。
  1. 關稅連鎖反應：2026 年美中貿易戰持續，若客戶將生產線移出中國大陸（影響中國 MLCC 需求），或對台廠加徵關稅（影響出口美國），都可能打亂國巨的全球通路布局。

  • 可控：成本管控、產品組合升級、客戶多元化（積極推進）
  • 不可控：景氣循環時機、關稅政策、原物料銀粉價格、日系廠競爭

1. 原物料假性需求風險（EP628）⚠️

  • 本波漲價有一部分驅動力來自原物料上漲（銀價大漲等）
  • 廠商必須轉嫁成本 → 下游怕更貴 → 提前備貨（Overbooking）
  • 雖然被動元件「易漲難跌」（像早餐店漲價回不去），但要分辨：
  • 終端需求推動（真需求，可延續）
  • 成本推動備貨（假需求，補完就停）
  • 警訊：若銀價回落後，訂單突然萎縮 → 代表本波部分是成本推動

2. 次族群復甦步調不一

  • 雖然電容族群非常缺，但電阻稼動率沒那麼高
  • 復甦排序：電容 > 電感 > 電阻
  • ⚠️ 不能無腦認為所有被動元件都會有同等爆發力
  • 對國巨影響：電阻營收占比越高，受益可能打折

3. priced-in 風險（EP652）⚠️

  • 台股資金效率極高 → 財報營收還沒出來，股價就把未來利多漲完了
  • 如果在相對高檔（如國巨現價 297）因新聞追進，可能承受較大回檔
  • 進場點偏高的情境：若進場價已接近 3 月高點（335 附近），此 priced-in 回檔風險更高，需搭配稼動率 / 族群資金判斷是否仍在題材初中段

4. 股癌觀察原則

  • 不要只看新聞利多追高
  • 看族群性資金是否還在匯聚（日本三大廠是否持續強）
  • 觀察稼動率是否仍在拉高 vs 開始走平（後者代表週期到頂）

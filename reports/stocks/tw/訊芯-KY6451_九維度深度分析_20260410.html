3. 估值面

股票類型判定

訊芯-KY 特殊情況：EPS 僅 0.26 元（近虧損），但 2026 年法人預估毛利率回升、業績放量，屬成長轉機型。

決策樹判定：近 4 季 EPS 合計偏低但轉正，且 2026 年確認成長趨勢 → 適用 TYPE_D（轉機股） 雙軌估值框架。

方法 A：PS 營收比法（TYPE_D 回顧型）

方法 B：技術面支撐法（TYPE_D 前瞻型）

雙軌估值計算

方法 A：PS 營收比法（回顧型）

白話說明：PS ratio = 市值 ÷ 年營收，用於虧損 / 轉機股。數字代表市場願意用幾倍的年營收來買這家公司。

```

數據：

  • 2025 年全年營收：75.32 億元
  • 流通股本：約 1.05 億股（市值 432 × 1.05 億 ≈ 454 億）
  • 每股年營收：75.32 億 ÷ 1.05 億股 ≈ 71.7 元

同業 PS ratio 參考：

  • 眾達-KY（4977）：市值約 213 億 / 年營收（需查）→ ❌ 精確 PS 未取得
  • 上詮（3363）：市值 87 億 / 年營收（需查）→ ❌ 精確 PS 未取得
  • 光通訊封裝同業 PS 中位數：業界估計約 4~6x（⚠️ 單一來源，WebSearch 估算）

合理計算：

fair_value_a = 同業 PS 中位數 5x × 每股年營收 71.7 元 = 358.5 元

```

⚠️ 降級說明：同業 PS 精確數值未能 WebFetch 取得，採業界估算中位數 5x，可信度「單一來源，建議自行覆核」。

方法 B：技術面支撐法（前瞻型）

```

數據（technical_indicators.py，4/10）：

  • MA60（季線）：214.93 元
  • MA120（半年線）：183.69 元
  • MA20（月線）：302.68 元

fair_value_b = MA120 × 技術面合理溢價（CPO 題材股通常享有半年線 1.3~1.5x 溢價）

= 183.69 × 1.4 = 257 元（⚠️ 技術面估值，僅供參考）

保守取值：fair_value_b = MA60 × 1.0 = 214.93 元（季線為中期支撐）

```

⚠️ 降級說明：方法 B 為純技術面，無基本面錨定，代表市場多空中性的均衡價格。

差距判定與進場區間

```

fair_value_a = 358.5 元（PS 法）

fair_value_b = 214.93 元（MA60 支撐）

差距 = |358.5 - 214.93| / 214.93 = 66.8%

→ 差距 > 40%，取較低者 + 標 ⚠️ 估值分歧大

anchor = min(358.5, 214.93) = 214.93 元

```

TYPE_D 進場區間：

```

entry_high = 214.93 × 0.90 = 193.4 元 ≈ 193 元

entry_low = 214.93 × 0.80 = 171.9 元 ≈ 172 元

```

技術面微調：現價 432 元遠高於 entry_high（193 元）超過 15%，標注「等回調」。

估值結論：

方法合理價公式
方法 A（PS 營收比法）358.5 元同業 PS 中位數 5x × 每股年營收 71.7 元

⚠️ 重要提醒：訊芯現在是「本夢比」定價模式——現價 432 元反映的是 2026~2027 年 CPO 業務大量放量、毛利率回升到 22%+ 的預期。若依傳統估值，現價明顯偏貴；但若 CPO 如期兌現，長期潛力仍大。投資人要問自己：「我是在買現在的公司，還是在賭未來的故事？」

1. 基本面

公司主要業務

鈊象電子（International Games System，IGS）成立於 1989 年，是台灣電子遊戲授權產業龍頭。

兩大業務線：

  • 網路遊戲（佔營收約 93%）：旗下 TaDa Gaming 平台將自研遊戲（麻將、老虎機、電子骰、捕魚機等）透過 B2B 授權模式，授權給全球各地的線上博弈平台，收取授權費與服務費。這是核心高毛利獲利引擎。
  • 商用遊戲機（佔約 7%）：傳統大型電玩機台，比重持續下滑。

產業鏈定位（ASCII 流程圖）

```

【遊戲開發商（鈊象）】

↓ 授權遊戲內容

【線上博弈平台商（全球代理商）】

↓ 吸引玩家

【終端玩家（全球各地）】

```

鈊象在「中游授權」位置，不直接面對玩家，不碰金流，風險極低，毛利極高。

獲利模式（護城河所在）

護城河 → 讓競爭對手很難搶走生意的優勢。

授權模式（Licensing）：不需直接經營賭場或面對玩家，只賣「遊戲內容版權」給平台商，收版稅。平台商吸引越多玩家 → 鈊象收到越多授權費。這是典型的「軟體版稅商業模式」，毛利率接近 98%，業界罕見。

護城河強度分析：

  • 技術積累：30 年以上遊戲研發，遊戲庫龐大（尤其亞洲文化主題如麻將、百家樂，難以複製）
  • 全球執照：已取得馬爾他、瑞典、希臘、羅馬尼亞、英國、葡萄牙等多國監管執照，門檻高
  • 客戶黏著：平台商一旦整合 TaDa Gaming 遊戲，切換成本高
  • 規模效應：遊戲開發是固定成本，授權的邊際成本接近零，規模越大利潤越高

地理分佈（更新版）

  • 海外授權佔整體營收約 68%，東南亞是最大市場（約 40%）
  • 台灣約 29%
  • 歐洲市場快速擴張中（英國、瑞典、希臘、葡萄牙、羅馬尼亞已取得執照並上線）
  • 南非 2026 年正式上線（非洲市場切入）
  • 美國市場為中長期目標

TaDa Gaming 戰略地位升級

2025 年最大亮點：TaDa Gaming 從過去第 6 大營收來源，升至今年第 2 大（超越《明星三缺一》），成長動能強，成為海外擴張的核心引擎。

客戶集中度

鈊象採 B2B 多元授權模式，客戶為全球博弈平台商，無單一大客戶依賴問題，客戶集中度低，風險分散。

管理層品質

  • 創辦人家族持股穩定，董監持股約 8%
  • 歷史上法說會的業績承諾與實際達成吻合度高（月營收連創新高是最佳佐證）
  • 無重大誠信疑慮

雙軌估值（TYPE_A 成長股）

鈊象 EPS 年增 20%+，屬成長股（TYPE_A），使用歷史 PE 區間法 + PEG 估值法。

方法 A（歷史 PE 區間法）：

```

使用 Forward EPS（成長股適用）：47.3 元

× 歷史 PE 中位數：20x

= 合理價 946 元

```

方法 B（PEG 估值法）：

```

EPS 年增率：22.8%（47.3/38.51 - 1）

合理 PE = 22.8（PEG = 1 時合理）→ 取整 23x

23x × Forward EPS 47.3 = 1,088 元

```

PEG → 考慮成長速度後，這檔股票到底貴不貴。PEG = 1 表示股價成長和獲利成長剛好匹配，是合理水準。

差距與錨定：

```

差距 = (1088 - 946) / 946 = 15% → 取平均

錨定價 = (946 + 1088) / 2 = 1,017 元

```

進場區間（TYPE_A）：

```

entry_high = 1,017 × 0.90 = 915 元

entry_low = 1,017 × 0.85 = 865 元

```

現價 769 元，低於進場區（865-915 元）約 11-16%。

→ 這代表現價已低於基本面估算的合理進場區，是加碼的基本面訊號，但仍需搭配技術面確認。

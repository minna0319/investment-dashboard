1. 基本面

公司簡介

華城電機成立於 1969 年，是台灣最具規模的電力變壓器製造商之一，專營大型與高壓電力變壓器。

變壓器 → 把發電廠產出的超高壓電，一路降壓到家庭和工廠能用的電壓。沒有它，電就送不到你家。

主要業務與營收結構

  • 變壓器：佔營收約 73%，核心產品，包括配電變壓器、電力變壓器、超高壓變壓器
  • 配電盤/開關設備：佔營收約 15-20%
  • 工程服務：佔營收約 5-10%

產業鏈定位

```

上游（原材料） 中游（製造） 下游（客戶）

矽鋼片/銅線 ──→ 【華城電機】 ──→ 台電/電力公司

絕緣材料 ──→ 變壓器製造 ──→ AI 資料中心

鋼板/零組件 ──→ ──→ 工業/商業用戶

──→ 美國電網業者

```

客戶結構

  • 台電：國內最大客戶，受惠台電「強韌電網計畫」→ 訂單持續釋出
  • 美國電力公司 / 雲端巨頭：外銷佔比 2025H1 已達 54%，星際之門計畫是重要客戶
  • 客戶集中度偏高（台電 + 美國客戶為兩大支柱），但因訂單能見度高達 2028 年，短期風險可控

競爭優勢（護城河）

護城河類型說明強度
技術門檻大型變壓器需要長期經驗累積，新廠商進入至少需 3-5 年中強
客戶認證台電與國際電力公司認證門檻高，已認證的供應商有先行者優勢強
產能卡位全球變壓器供不應求，交期延長至 2-3 年，華城產能已被預訂強（短期）
規模經濟台灣重電四雄之一，但全球來看規模仍遠小於 GE Vernova、Siemens Energy中
護城河 → 讓競爭對手很難搶走生意的優勢。華城的護城河主要來自客戶認證難拿、產能被搶光這兩點。

管理層

  • 經營團隊穩定，近年積極拓展海外市場
  • 2025 年啟動觀音廠與台中港廠擴建計畫，顯示對未來需求有信心
  • 台中港廠斥資 25 億元，預計 2027Q2 投產，可生產 500KV 以上超高壓變壓器

3. 估值面

股票類型判定

  • EPS YoY > 15%（2026E 成長 55%）→ TYPE_A 成長股

方法 A：歷史 PE 區間法（回顧型）

本益比（PE） → 用目前股價除以每股盈餘，代表你花多少錢買 1 元的獲利。PE 20 倍 = 花 20 元買 1 元利潤。

華城近 4 年 EPS：1.11 / 3.21 / 9.87 / 14.93 = 平均 7.28 元

但華城處於高速成長期，過去 EPS 已不代表未來，因此改用 Forward EPS（法人預估 2026E EPS = 21.80 元）。

華城目前 PE 約 46 倍（以 TTM EPS 計算）。由於華城從低基期快速成長，歷史 PE 區間波動極大（從 10 倍到 80 倍都有），參考法人共識：

```

合理 PE 區間（成長股）：35-45 倍

fair_value_a = 21.80 × 40 = 872 元

```

方法 B：PEG 估值法（前瞻型）

PEG → 「考慮成長性之後，這檔股票到底貴不貴」。PEG < 1 = 成長被低估，PEG = 1 = 合理。

```

EPS 年成長率 = 55.8%（2025→2026E）

合理 PE = 30（TYPE_A 上限 30x，因成長率極高但取保守值）

Forward EPS = 21.80

fair_value_b = 30 × 21.80 = 654 元

目前 PEG = 46.37 ÷ 55.8 = 0.83 → 成長性角度看偏低估

```

差距判定與進場區間

```

差距 = |872 - 654| / 654 = 33.3% → 21-40%，取較低者

anchor = 654 元（取方法 B）

anchor_rule = "取較低(方法B)"

```

進場區間（TYPE_A：anchor × 0.85 ~ 0.90）：

```

entry_low = 654 × 0.85 = 556 元

entry_high = 654 × 0.90 = 589 元

```

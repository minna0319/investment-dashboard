1. 基本面

公司簡介

建準電機（2421）成立於 1980 年，是全球第二大散熱風扇製造商，總部位於高雄。用一句話說：建準靠做「幫電子設備散熱」的風扇和模組賺錢。

產業鏈定位

```

上游（零組件/材料） → 中游（建準） → 下游（品牌/組裝廠）

馬達、軸承、塑膠 風扇模組設計製造 伺服器廠（廣達、緯穎）

PCB、散熱片 液冷散熱模組 網通設備商（思科）

銅管、水冷板材料 幫浦/Cold Plate IT 品牌（Dell、HP）

車廠（電動車散熱）

```

建準位於中游，核心能力是把散熱元件整合成完整模組，再賣給下游組裝廠和品牌商。

營收結構（2025Q3 基準）

重點：AI 伺服器在 2025Q3 單季營收 11.35 億元，佔比已達 23.5%，是最大成長引擎。

客戶集中度

建準服務的客戶分散在伺服器、網通、車用等多個領域，單一客戶依賴度不算特別高，但 NVIDIA 供應鏈認證（NVL72、GB200、GB300）是關鍵，若失去認證將是風險。

  • 近年策略方向明確：從純風扇轉型為「氣冷 + 液冷整合方案供應商」
  • 菲律賓擴廠（未來兩年投入 20 億元）、新北研發中心規劃，顯示管理層有積極擴張意圖

3. 估值面

股票類型判定

  • EPS YoY +45%（> 15%）→ TYPE_A 成長股
  • 配對方式：方法 A（歷史 PE 區間法）+ 方法 B（PEG 估值法）

方法 A：歷史 PE 區間法（回顧型）

目前建準 PE 約 17.74 倍（以最新收盤價 141 元 ÷ 近四季 EPS 7.94 元）。

參考歷年 PE 區間（2022-2025）：

  • 2022 年股價區間約 36-63 元，EPS 4.34 → PE 約 8-15x
  • 2023 年股價區間約 50-88 元，EPS 5.16 → PE 約 10-17x
  • 2024 年股價區間約 64-176 元，EPS 5.46 → PE 約 12-32x（AI 題材推升）
  • 2025 年股價區間約 64-190 元，EPS 7.94 → PE 約 8-24x
PE 參數數值說明
歷史 PE 低標（25th）~12x市場悲觀時的估值
歷史 PE 中位數~17x市場中性估值
歷史 PE 高標（75th）~24x市場樂觀時的估值

使用 Forward EPS（2026E）10.8 元計算（成長股用前瞻 EPS 更合理）：

```

fair_value_a = Forward EPS 10.8 × 歷史 PE 中位數 17x = 183.6 元

便宜價 = 10.8 × 12x = 129.6 元

昂貴價 = 10.8 × 24x = 259.2 元

```

方法 A 合理價：183.6 元

方法 B：PEG 估值法（前瞻型）

PEG → 考慮成長性之後，股票到底貴不貴。PEG < 1 表示成長被低估。

```

EPS 年成長率 = 36%（2026E vs 2025）

合理 PE = 36 × 1（PEG = 1）= 30x → 但 TYPE_A 上限 30x，取 30x

fair_value_b = 30x × Forward EPS 10.8 = 324 元

```

但 PEG 算出的值偏高，需要保守處理。考量建準並非純 AI 股（AI 佔比約 25-30%），用 PEG = 1 偏樂觀。調整為 PEG = 0.7（考慮部分傳統業務拉低整體成長）：

```

合理 PE = 36 × 0.7 = 25.2x

fair_value_b = 25.2 × 10.8 = 272.2 元

```

方法 B 合理價：272.2 元

差距判定與進場區間

```

差距 = |183.6 - 272.2| / 183.6 = 48.3%

```

差距 > 40% → 取較低者 + ⚠️ 估值分歧大

```

anchor = 183.6（方法 A）

```

TYPE_A 進場區間：

```

entry_high = 183.6 × 0.90 = 165.2 → 取整 165 元

entry_low = 183.6 × 0.85 = 156.1 → 取整 156 元

```

技術面微調

  • MA120（半年線）= 149.9 元，entry_low 156 元距離 MA120 約 +4%
  • 現價 141 元低於 entry_low 156 元 → 目前股價已在合理進場區間以下
  • 但技術面偏弱（死亡交叉），不宜急著接刀

估值面結論

```

方法 A（歷史PE區間法）：合理價 183.6 元（Forward EPS 10.8 × 歷史PE中位數 17x）

方法 B（PEG估值法）：合理價 272.2 元（Forward EPS 10.8 × 合理PE 25.2x, PEG=0.7）

差距：48.3% → 取較低(方法A) + ⚠️ 估值分歧大

錨定價：183.6 元

進場區間：156 ~ 165 元

備註：⚠️ 估值分歧大（方法B明顯高於方法A），建議以方法A為主要參考。現價141元低於進場區間，技術面偏弱宜等回穩。

```

白話說：用歷史估值看，建準合理價約 184 元，目前 141 元確實偏便宜。但估值分歧大，市場對未來成長力道看法不一，需要更多營收數據確認 2026 年成長是否如預期。

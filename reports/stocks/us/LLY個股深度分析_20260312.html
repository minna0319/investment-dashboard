1. 基本面

公司簡介

Eli Lilly（禮來）是全球前五大製藥公司，1876 年創立於美國印第安納州，目前市值接近 1 兆美元。

一句話說清楚：禮來靠賣「治療糖尿病 + 減重」的明星藥品賺錢，目前是全球 GLP-1 藥物市場的龍頭。

3. 估值面

目前估值指標

指標數值解讀
Trailing P/E~45x高於大盤但低於自身歷史均值
Forward P/E（2026E）~29x考慮成長後更合理
3 年平均 P/E~83x目前大幅低於歷史均值
5 年平均 P/E~67x仍低於歷史均值
PEG~0.7（估算）< 1 → 成長性被低估 ✅
P/E（本益比） → 用幾年的獲利能回本。45 倍 = 假設獲利不變，要 45 年才回本。但如果獲利快速成長，實際回本時間會短很多。
PEG → 「考慮成長性之後，到底貴不貴」。PEG < 1 表示成長速度超過了股價的溢價程度，算是被低估。

雙軌估值計算

類型判定：TYPE_C（美股科技/成長型製藥），EPS 高速成長 > 15%

方法 A：歷史 PE 區間法（回顧型）

```

近 4 年平均 EPS ≈ ($6.32 + $13.0 + $24.21) / 3（2023 前 EPS 較低，取近 3 年加權）

→ 考慮到 EPS 成長極快，用 Forward EPS 更合理

Forward EPS（2026E）= $34.25（指引中位數）

歷史 PE 低標（近 5 年 25th）≈ 40x

歷史 PE 中位數 ≈ 55x

fair_value_a = Forward EPS $34.25 × 歷史 PE 低標 40x = $1,370

（用 PE 低標而非中位數，已是保守取法）

但 TYPE_C 上限 40x，所以取 40x

fair_value_a = $34.25 × 40 = $1,370

```

方法 B：PEG 估值法（前瞻型）

```

EPS 年成長率 ≈ 45%（2026E $34.25 vs 2025 $24.21）

→ 但高成長不可能永續，取保守成長率 30%

合理 PE = 30（PEG = 1）

→ TYPE_C 上限 40x，30x 在範圍內

fair_value_b = Forward EPS $34.25 × 30 = $1,028

```

差距判定

```

差距 = |$1,370 - $1,028| / $1,028 = 33.3%

→ 21-40% → 取較低者

anchor = $1,028（方法 B）

```

進場區間（TYPE_C：anchor × 0.82 ~ 0.88）

```

entry_low = $1,028 × 0.82 = $843

entry_high = $1,028 × 0.88 = $905

```

技術面微調

  • MA200 = $883，在 entry_low ($843) 和 entry_high ($905) 之間
  • MA200 $883 在 entry_high $905 的 ±3% 內 → 吸附到 $880
  • 現價 $1,000 高於 entry_high $905 約 10.5% → 備註「需等回調」

估值小結

```

方法 A（歷史 PE 區間法）：合理價 $1,370（Forward EPS $34.25 × 40x）

方法 B（PEG 估值法）：合理價 $1,028（Forward EPS $34.25 × 30x）

差距：33.3% → 取較低者（方法 B）

錨定價：$1,028

進場區間：$843 ~ $905（技術面微調後 $843 ~ $880）

備註：現價 $1,000 高於進場上緣約 10%，需等回調

```

催化劑解讀

  • 最重要的催化劑是 Orforglipron FDA 核准（4/10 目標日）：如果通過，等於打開「口服減重藥」的龐大市場（很多人怕打針不願用 Zepbound），股價可能跳漲

禮來是一家很好的公司，GLP-1（減重/糖尿病藥）這個賽道未來幾年成長空間巨大。但「好公司」不等於「好價格」——目前 $1,000 的股價已經反映了大部分利多。

結論

【觀望】— Iridium 是衛星通訊領域具備獨特護城河的現金牛，但 2026 年成長放緩（指引僅 0-2%）、高負債是主要隱憂，目前股價已反映基本面，建議等回調至合理估值再進場。

1. 基本面

一句話說清楚

Iridium 靠 66 顆低軌衛星（LEO）組成的全球唯一「極地到極地」通訊網路，提供語音、數據、IoT（物聯網）和政府安全通訊服務來賺錢。

產業鏈定位

```

上游：衛星製造（Thales Alenia Space）、火箭發射（SpaceX）

↓

中游：【Iridium】衛星網路營運商（擁有+營運 66+14 顆衛星）

↓

下游：終端設備商 → 企業客戶（海運/航空/採礦）、政府（美軍/國防）、消費者

```

營收結構（2025 全年）

類別金額佔比趨勢

設備銷售 & 工程支援~$2.38 億~27%波動較大
合計$8.72 億100%YoY +5%

服務收入細分（核心成長引擎）：

  • 商業 IoT：最大成長動力，物聯網裝置連接數持續增加
  • 商業語音與數據：海事、航空等行業穩定需求
  • 美國政府：國防與安全通訊合約（EMSS），穩定但集中度高
  • 託管酬載（Hosted Payload）：衛星上額外搭載的政府設備

客戶集中度

⚠️ 美國政府佔比偏高（具體比例未揭露，但公司明確提示這是風險因子）。失去政府合約會造成重大影響。

護城河 → 讓競爭對手很難搶走生意的優勢。Iridium 的護城河在於它是全球唯一能在地球任何角落通訊的衛星網路，包括南北極和公海。

  • 積極回饋股東（股利+回購），但 2026 指引偏保守，顯示管理層務實

3. 估值面

類型判定

  • EPS YoY +23%（>15%）→ TYPE_A 成長股（但成長率預計放緩）
  • 同時有殖利率 2.4% → 介於 A 和 B 之間，以 TYPE_A 為主

雙軌估值

方法 A：歷史 PE 區間法（回顧型）

  • 近 4 年平均 EPS：Iridium 近年 EPS 波動大（2022 年 $0.04、2023 $0.52、2024 $0.86、2025 $1.06），取近 2 年較穩定值平均 = ($0.86 + $1.06) / 2 = $0.96
  • 歷史 PE 區間：由於歷史 PE 極度波動（5x ~ 1,109x），使用近 2 年穩定獲利期的 PE，目前 trailing PE ~15.7x，參考同業中位數 ~20x
  • PE 低標（保守）：15x
  • PE 中位數：20x

```

fair_value_a = $0.96 × 20 = $19.20

```

⚠️ 方法 A 備註：Iridium 歷史 PE 極度波動，近 2 年才進入穩定獲利期，歷史 PE 參考價值有限，部分引用同業中位數。

方法 B：PEG 估值法（前瞻型）

  • Forward EPS 預估：Wall Street 預估 2026 EPS 成長 ~15% → Forward EPS ≈ $1.06 × 1.15 = $1.22
  • EPS 成長率：15%
  • 合理 PE（PEG=1）：15x
  • PEG 現值：trailing PE 15.7 / EPS 成長率 15 = 1.05（接近合理）

```

fair_value_b = $1.22 × 15 = $18.30

```

差距判定

```

差距 = |$19.20 - $18.30| / $18.30 = 4.9% → ≤ 20%

取值方式：取平均

anchor = ($19.20 + $18.30) / 2 = $18.75

```

進場區間（TYPE_A）

```

entry_high = $18.75 × 0.90 = $16.88

entry_low = $18.75 × 0.85 = $15.94

```

Starlink（SpaceX，未上市）

  • 規模壓倒性：9,400+ 顆衛星 vs Iridium 66 顆
  • 速度差距大：Starlink 50-250 Mbps vs Iridium 88 kbps（差距 1,000 倍以上）
  • 但不完全競爭：Starlink 主打寬頻上網，Iridium 主打窄頻 IoT + 緊急通訊 + 政府安全通訊，兩者核心市場不同
  • 威脅：海事客戶已開始將 Iridium 從「主要」降為「備援」服務

同業比較結論

Iridium 在同業中是最穩健的選擇：已獲利、有現金流、有護城河。但市值相對低（因成長慢），而市場更偏好高成長的 ASTS。若追求穩健，IRDM 優於 GSAT 和 ASTS；若追求爆發力，IRDM 不是首選。
